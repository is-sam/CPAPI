/lang/commodities get

get

Successfully retrieve all available commodities

get /lang/commodities

Successfully retrieve all available commodities

Secured by oauth_2_0

OAuth2 is a protocol that lets external apps request authorization to access the API

  • Request
  • Response

Headers

  • Authorization: required (string)

    Used to send a valid OAuth 2 access token. Do not use together with the "access_token" query string parameter.

Query Parameters

  • access_token: required (string)

    Used to send a valid OAuth 2 access token. Do not use together with the "Authorization" header

  • country: (string)

    code of the country

    Example:

    GB
  • flow: (string)

    used to describe the type of flow wanted, 'i' for import, 'e' for export, 'all' for both types

  • from_date: (string)

    starting date for the query, date format: YYYY-MM

    Example:

    2012-04
  • to_date: (string)

    end date for the query, date format: YYYY-MM

    Example:

    2012-04

/access_token

/access_token post

post

Retrieve an access token in order to access the API

post /access_token

Retrieve an access token in order to access the API

Secured by oauth_2_0

OAuth2 is a protocol that lets external apps request authorization to access the API

  • Request
  • Response

Headers

  • grant_type: required (string)

    grant_type, value='password'

  • client_id: required (string)

    Client Identification obtained on registration

  • client_secret: required (string)

    Client Secret obtained on registration

  • username: required (string)

    Client email

  • password: required (string)

    Client password

Body

Type: application/json

HTTP status code 200

Successfully created a new token

Body

Type: application/json

Example:

{
    "access_token": "IMhaIdD9iiQAP0OZkehVCn4HvI42RCBK3XHreT8S",
    "token_type": "Bearer",
    "expires_in": 3600
}
